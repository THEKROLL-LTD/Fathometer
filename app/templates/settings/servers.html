Add a server

Run the installer on the target host as root. It installs Trivy and the agent, registers the host with the master key (asked interactively), and sets up a daily scan timer. No UI onboarding needed. 

sudo bash <(curl -fsSL {{ external_base_url }}/install.sh)

The server key never leaves the host. To rotate a host’s key, just re-run the installer on that host.

To remove the agent from a host, run sudo /opt/fathometer/bin/fathometer-uninstall.sh on it (or sudo bash <(curl -fsSL {{ external_base_url }}/uninstall.sh)). This is local-only — delete the host below to drop it from the dashboard.
